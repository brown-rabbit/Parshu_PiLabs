# Parshu_PiLabs

A collection of Raspberry Pi homelab experiments built as part of Project Parshuram.

This repository documents small practical projects focused on:

- Linux server administration
- Networking fundamentals
- Self-hosted services
- Beginner cybersecurity labs

The goal is to learn by building real infrastructure and documenting the process.

# Lab Environment

Hardware:
- Server Device: Raspberry Pi 4  
            RAM: 4 GB  
        Storage: MicroSD Card

- Laptop: Lenovo IdeaPad Gaming 3i  
    CPU: Intel Core i5 (10th Gen)  
    GPU: NVIDIA GTX 1650  
    RAM: 16 GB  
    Storage: 512 GB SSD  

Operating System:
- Laptop: Windows 11  
- Server: Raspberry Pi OS

# Lab Projects

 Project  Description 
|-------|------------|
| Tailscale Remote Access  (Securely access the Raspberry Pi from any network) |
| Pi-hole DNS Server  (Network-level ad blocking using a local DNS server) |
| Nginx Web Server (Hosting a basic website from the Raspberry Pi) |
| AWS Cloud Static Website (Deploy a simple static website in the cloud using Amazon Web Services Amazon S3 free tier) |
| Future Labs (Honeypots, log monitoring, automation) |
